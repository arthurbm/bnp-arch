## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues for `arthurbm/bnp-arch`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses the default five-label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Domain documentation uses a single-context layout with root `CONTEXT.md` and root `docs/adr/` when present. See `docs/agents/domain.md`.

### Design sessions (grilling)

When presenting design options, every option must be a genuine candidate with defensible trade-offs — no strawman options that only validate the recommendation. If an option has no honest defense, don't list it; two strong options beat three with filler.
